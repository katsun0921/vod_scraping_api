# VOD配信情報収集パイプライン 改善提案

作成日: 2026-07-23
対象: [vod-release-calendar-spec.md](vod-release-calendar-spec.md)

---

## 目的

現行仕様は完成度が高く、VOD配信情報を収集・承認・WordPress・SNSへ展開する一連の
ワークフローとして十分実用的である。

一方、Katsumascoreが目指す「映画レビューサイト」ではなく**映画・アニメ総合メディア**
としてのブランド価値を高めるためには、単なる配信情報の集約に留まらず、既存レビュー資産や
データベースとの連携を強化することが重要となる。

本書では、ブランド価値向上の観点から改善案を整理する。

---

## 1. 編集部視点（Editor's Pick）の追加【優先度：S】

### 現状

週次記事は配信作品を一覧表示する構成となっている。

```
Netflix

作品A

作品B

Prime Video

作品C
```

情報としては十分だが、ニュースの羅列になりやすい。

### 改善案

記事冒頭に「編集部おすすめ作品」を設置する。

例:

```
編集部おすすめ

★★★★★
インターステラー

今週配信作品の中でも特におすすめ。
宇宙SFの名作として現在も高い評価を受けている作品。

レビューはこちら
```

### 効果

- メディア独自性が生まれる
- ブランド価値向上
- レビュー記事への導線強化
- AIではなく編集者が選ぶ価値を提供できる

---

## 2. レビュー記事への送客強化【優先度：S】

### 現状

レビュー記事が存在する場合のみ内部リンクを掲載する。

### 改善案

作品カードを統一する。

例:

```
作品タイトル

配信開始日

配信サービス

★★★★☆

レビューを読む

作品詳細を見る
```

### 効果

- レビューPV増加
- 関連記事回遊率向上
- 作品データベース活用

---

## 3. サービス別ページとの連携【優先度：S】

将来的に

```
Netflix

Prime Video

Disney+

U-NEXT
```

などサービスページを作成する。

### ページ構成

```
今週追加作品

今月追加作品

人気レビュー

関連記事

最新ニュース
```

### 効果

- サービス名検索を獲得
- カテゴリページとしてSEO強化

---

## 4. 年間アーカイブの自動生成【優先度：A】

週次記事を蓄積し

```
2026年 Netflix追加作品

2026年 Prime Video追加作品

2027年 Disney+追加作品
```

などを自動生成する。

### 効果

- ロングテールSEO
- 過去配信作品検索
- 資産化

---

## 5. ジャンル別まとめページ【優先度：A】

Movieデータを利用して

```
今週追加されたSF映画

今週追加されたホラー映画

今週追加されたアニメ

今週追加された日本映画
```

などを自動生成する。

### 効果

- 検索流入増加
- Movieデータベース活用

---

## 6. 作品ページへの逆リンク【優先度：A】

作品ページへ

```
最新配信情報

Netflix

2026/7/29配信開始
```

を表示する。

### 効果

ニュース→作品、作品→ニュースの双方向リンクが形成される。

---

## 7. CPT名称の見直し【優先度：B】

現在

```
vod_news
```

となっている。

実際にはニュースではなく

- 配信予定
- 配信カレンダー
- 配信一覧

として利用される。

候補:

```
vod_calendar

vod_release

vod_schedule
```

### 理由

ニュースは速報も含むが、本コンテンツは予定情報・配信情報を整理したアーカイブである。

## 8. 承認フロー改善【優先度：B】

現在

```
Slack

↓

Google Sheets

↓

承認

↓

WordPress
```

となっている。

将来的には

```
Slack通知

↓

承認ボタン

↓

Google Sheets更新

↓

WordPress投稿
```

のようなワンクリック承認へ発展できる。

### 効果

- 運用工数削減
- 人的ミス防止

---

## 9. Google Discover対策【優先度：S】

記事構成を

```
編集部おすすめ

↓

今週追加作品

↓

サービス別まとめ

↓

レビューリンク

↓

関連記事
```

に統一する。

### 効果

- Google Discoverとの親和性向上
- 内部リンク増加
- ブランド価値向上

---

## 10. ブランド価値向上

本施策は単なるVOD配信情報の掲載ではなく、

```
配信開始

↓

レビュー

↓

人物

↓

シリーズ

↓

関連記事
```

という導線を作ることが目的となる。

これにより「レビューサイト」から「映画・アニメ総合メディア」へブランドを拡張できる。

## 総括

現行仕様は、VOD配信情報を収集・配信するパイプラインとして高い完成度を持つ。

今後は「情報を集める」ことに加え、「既存レビュー資産・作品データベース・人物ページ・
シリーズページへ読者を送客するハブ」として発展させることで、Katsumascoreのブランド価値を
さらに高められる。

VOD配信情報は、映画・アニメファンが作品と出会う入口となるコンテンツであり、
レビュー・考察・作品データベースを結び付ける中心的な役割を担う施策として位置付ける。
